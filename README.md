## ✨ Features

- 🔐 **JWT Authentication** — Secure login with cookie-based session
- 🛒 **Smart Cart** — Add, remove, increment, decrement with real-time total
- 🔍 **Keyword Search** — Search products instantly
- 🎛️ **Multi-Filter** — Filter by category AND rating simultaneously
- 🔒 **Protected Routes** — Cart accessible only after login
- 🌐 **Context API** — Global cart state management
- 📱 **Fully Responsive** — Bootstrap responsive grid
- 🔄 **REST APIs** — Full CRUD operations

## 🚀 How to Run Locally

```bash
# Clone the repository
git clone https://github.com/goulipsharathkumar/NxtTrend-web.git

# Navigate to project folder
cd NxtTrend-web

# Install dependencies
npm install

# Start the development server
npm start
```

Open http://localhost:3000 in your browser.
