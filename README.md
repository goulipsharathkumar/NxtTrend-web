# NxtTrendz – E-Commerce Web App 🛍️

A full-featured e-commerce Single Page Application built with React.js, 
featuring product browsing, cart management, JWT authentication, 
and protected routing.

## 🌐 Live Demo
👉 https://nxt-trend-web.vercel.app

## 📌 GitHub Repository
👉 https://github.com/goulipsharathkumar/NxtTrend-web

## 🚀 Features

- 🔐 JWT Authentication — secure login with Cookie-based session management
- 🛡️ Protected Routes — unauthenticated users redirected to Login
- 🛍️ Product Listing — browse products with keyword search and multi-filter
- 🔍 Filters — filter by category and rating simultaneously
- 🛒 Cart Management — add, remove, increment, decrement cart items
- 💰 Cart Summary — real-time total amount and item count
- 🗑️ Remove All — clear entire cart with one click
- 📱 Fully Responsive — works on mobile, tablet and desktop
- ⚡ Reusable Components — ProductCard, FilterGroup, SearchBar, CartItem

## 🛠️ Tech Stack

- React.js — Frontend library
- React Router DOM — Client-side routing
- Context API — Global cart state management
- JWT Authentication — Secure token based login
- JS Cookie — Cookie based session management
- REST APIs — Data fetching with Axios
- Bootstrap — Responsive styling
- CSS3 — Custom styling
